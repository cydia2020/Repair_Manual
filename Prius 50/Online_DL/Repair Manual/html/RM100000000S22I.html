Title: THEFT DETERRENT / KEYLESS ENTRY: IMMOBILISER SYSTEM: B2784; Antenna Coil Open / Short; 2016 - 2022 MY Prius [11/2015 -        ]

DTC

B2784

Antenna Coil Open / Short

DESCRIPTION

When an open or short circuit is detected in the transponder key amplifier coil built into the power switch, the certification ECU (smart key ECU assembly) stores this DTC. This DTC is also stored as a history DTC.

DTC No.

Detection Item

Trouble Area

Note

B2784

Antenna Coil Open / Short

The transponder key amplifier coil built into the power switch is open (see below) or shorted (determined by communication with certification ECU (smart key ECU assembly)) (1 trip detection logic*).

  • Terminal AGND, VC5, TXCT or CODE circuit is open (including disconnection of the power switch).
  • The transponder key amplifier coil built into the power switch is open.
  • Power switch
  • Wire harness or connector
  • Certification ECU (smart key ECU assembly)

DTC output confirmation operation:

  • With the park (P) selected and the electrical key transmitter sub-assembly held near the power switch, a hybrid control system start operation is performed by pressing and holding the power switch when the transmitter battery is depleted.
  • *: Only output while a malfunction is present.

Vehicle Condition and Fail-safe Operation when Malfunction Detected

Vehicle Condition when Malfunction Detected

Fail-safe Operation when Malfunction Detected

Hybrid control system cannot be started when transmitter battery is depleted by holding electrical key transmitter sub-assembly near power switch and pressing and holding power switch with park (P) selected
