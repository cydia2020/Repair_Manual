INSTALLATION

PROCEDURE

1. INSTALL FRONT SEAT CUSHION AIRBAG ASSEMBLY RH

(a) Pass the airbag wire harness through the hole.

(b) Install the front seat cushion airbag assembly RH with 3 new nuts.

Torque:

5.5 N·m {56 kgf·cm, 49 in·lbf}

NOTICE:

  • The nuts must not be reused.
  • Make sure that the front seat adjuster assembly RH is not deformed. If it is, replace it with a new one.

(c) Engage the claw and 2 clamps.

HINT:

Engage the claw first.
