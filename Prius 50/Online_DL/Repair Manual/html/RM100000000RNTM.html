Title: CRUISE CONTROL: MILLIMETER WAVE RADAR SENSOR: REASSEMBLY; 2016 - 2020 MY Prius [11/2015 - 07/2020]

REASSEMBLY

CAUTION / NOTICE / HINT

NOTICE:

  • Do not damage the millimeter wave radar sensor assembly.
  • If the No. 1 millimeter wave radar sensor bracket has been removed/installed, be sure to perform adjustment.

PROCEDURE

1. INSTALL NO. 1 MILLIMETER WAVE RADAR SENSOR BRACKET

(a) Temporarily install the No. 1 millimeter wave radar sensor bracket with the 2 screws.

(b) Tighten the 2 screws to install the No. 1 millimeter wave radar sensor bracket.

HINT:

Tighten the screws in the order shown in the illustration.
