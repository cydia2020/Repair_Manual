Title: INTERIOR PANELS / TRIM: REAR CONSOLE BOX: INSTALLATION; 2016 - 2018 MY Prius [11/2015 - 12/2018]

INSTALLATION

PROCEDURE

1. INSTALL REAR CONSOLE BOX ASSEMBLY

(a) Install the rear console box assembly with the 6 bolts.

Torque:

13 N·m {133 kgf·cm, 10 ft·lbf}

2. INSTALL CONSOLE REAR END PANEL SUB-ASSEMBLY

(a) Connect the connector.

(b) Engage the 8 clips to install the console rear end panel sub-assembly as shown in the illustration.

Install in this Direction

3. INSTALL CONSOLE CUP HOLDER BOX SUB-ASSEMBLY

(a) Install the console cup holder box sub-assembly with the 6 screws.

4. INSTALL UPPER CONSOLE PANEL SUB-ASSEMBLY

(a) Connect each connector.

(b) Engage each clamp.

(c) Engage the 2 guides and 12 clips to install the upper console panel sub-assembly as shown in the illustration.

Install in this Direction

5. INSTALL NO. 1 UPPER BOX PLATE

(a) Engage the 2 claws as shown in the illustration.

Install in this Direction

(b) Engage the 2 clips and claw to install the No. 1 upper box plate as shown in the illustration.

Install in this Direction
