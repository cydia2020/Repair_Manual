INSTALLATION

PROCEDURE

1. INSTALL SLIDING ROOF HOUSING ASSEMBLY

(a) Loosen the 12 bolts of the brackets of the sliding roof housing assembly.

(b) Temporarily install the sliding roof housing assembly with the 6 nuts and 8 bolts.

(c) Tighten the 2 nuts.

HINT:

Tighten the 2 nuts in the order shown in the illustration.

Torque:

8.0 N·m {82 kgf·cm, 71 in·lbf}

(d) Tighten the 4 nuts.

Torque:

8.0 N·m {82 kgf·cm, 71 in·lbf}

(e) Tighten the 20 bolts to install the sliding roof housing assembly.

Bolt (A)

Bolt (B)

HINT:

The brackets can be installed in any order. The bolts must be tightened in the order of (A) then (B).

Torque:

8.0 N·m {82 kgf·cm, 71 in·lbf}

2. CONNECT SLIDING ROOF DRAIN HOSE ASSEMBLY LH

(a) Connect the sliding roof drain hose assembly LH.

HINT:

Slide the hose to the base of the drain pipe.

(b) Engage the claw to secure the sliding roof drain hose assembly LH.

HINT:

  • Make sure that the clamp is on the marking or between the marking and hose end.
  • Make sure that the lock of the clamp is facing up.

3. CONNECT SLIDING ROOF DRAIN HOSE ASSEMBLY RH

HINT:

Use the same procedure as for the LH side.

4. CONNECT SLIDING ROOF DRAIN HOSE

HINT:

  • Use the same procedure as for the sliding roof drain hose assembly LH.
  • Use the same procedure for both sliding roof drain hoses.

(a) Using a T25 "TORX" socket wrench, temporarily install the sliding roof glass sub-assembly with the 4 screws.

6. CHECK FOR WATER LEAK

(a) After adjusting the sliding roof glass sub-assembly, check for water leakage into the vehicle interior.

(b) If there are any leaks, readjust the sliding roof glass sub-assembly.
