PROCEDURE

3. INSTALL FRONT TURN SIGNAL LIGHT BULB

(a) Install the front turn signal light bulb to the front turn signal light socket.

(b) Turn the front turn signal light socket with the front turn signal light bulb as shown in the illustration to install them as a unit.

Install in this Direction
