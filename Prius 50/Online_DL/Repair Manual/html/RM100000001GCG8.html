Title: HYBRID / BATTERY CONTROL: FUSE BOX (for NICKEL METAL HYDRIDE BATTERY): INSTALLATION; 2019 - 2022 MY Prius [12/2018 -        ]

INSTALLATION

PROCEDURE

1. INSTALL HYBRID BATTERY TERMINAL BLOCK

CAUTION:

Be sure to wear insulated gloves and protective goggles.

(a) Connect the 2 hybrid battery terminal block connectors.

NOTICE:

Make sure that the connectors are connected securely.

(b) Install the hybrid battery terminal block to the HV battery with the bolt.

Torque:

7.5 N·m {76 kgf·cm, 66 in·lbf}

NOTICE:

Make sure to align the hole of the hybrid battery terminal block with the pin.
