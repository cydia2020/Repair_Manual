Title: CRUISE CONTROL: DYNAMIC RADAR CRUISE CONTROL SYSTEM: P057162; Brake Switch "A" Signal Compare Failure; 2016 - 2020 MY Prius [11/2015 - 07/2020]

DTC

P057162

Brake Switch "A" Signal Compare Failure

DESCRIPTION

When the brake pedal is depressed, the stop light switch assembly outputs a signal to the hybrid vehicle control ECU. When the hybrid vehicle control ECU receives the signal, it cancels control of vehicle speed by the dynamic radar cruise control system. The fail-safe function operates to enable normal operation even if there is a malfunction in the stop light signal circuit. The cancellation condition occurs when voltage is applied to terminal STP. When the brake pedal is depressed, voltage is applied to terminal STP of the hybrid vehicle control ECU through the STOP fuse and the stop light switch assembly, and the hybrid vehicle control ECU turns the dynamic radar cruise control system off.
