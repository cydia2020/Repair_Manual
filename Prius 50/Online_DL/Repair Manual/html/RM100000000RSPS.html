HINT:

2. REMOVE COURTESY LIGHT BULB

(a) Using a screwdriver with its tip wrapped with protective tape, disengage the claw and remove the courtesy light housing with the courtesy light cover as shown in the illustration.

Protective Tape

Remove in this Direction

(c) Remove the courtesy light bulb from the courtesy light housing as shown in the illustration.

Remove in this Direction

3. INSTALL COURTESY LIGHT BULB

(a) Install the courtesy light bulb to the courtesy light housing as shown in the illustration.

Install in this Direction

(b) Install the courtesy light cover to the courtesy light housing as shown in the illustration.

(c) Engage the claw to install the courtesy light housing with the courtesy light cover as shown in the illustration.
