Title: SEAT BELT: FRONT SEAT INNER BELT ASSEMBLY: INSTALLATION; 2016 - 2022 MY Prius [11/2015 -        ]

INSTALLATION

CAUTION / NOTICE / HINT

HINT:

  • Use the same procedure for the RH side and LH side.
  • The following procedure is for the LH side.

PROCEDURE

1. INSTALL FRONT SEAT INNER BELT ASSEMBLY (for Driver Side)

(a) Install the front seat belt anchor plate as shown in the illustration.

Install in this Direction

(b) for Manual Seat:

(1) Install the front seat inner belt assembly with the nut.

Torque:

42 N·m {428 kgf·cm, 31 ft·lbf}

NOTICE:

Do not allow the anchor part of the front seat inner belt assembly to overlap the protruding part of the separate type front seat cushion spring assembly.

*a

Protruding Part

(2) Engage the 2 clamps.

(c) for Power Seat:

(1) Install the front seat inner belt assembly with the nut.

Torque:

42 N·m {428 kgf·cm, 31 ft·lbf}

NOTICE:

Do not allow the anchor part of the front seat inner belt assembly to overlap the protruding part of the separate type front seat cushion spring assembly.

*a

Protruding Part

(2) Engage the 2 clamps.

(d) Engage each clamp.

(e) Connect each connector.

2. INSTALL FRONT SEAT INNER BELT ASSEMBLY (for Front Passenger Side)

(a) Install the front seat belt anchor plate as shown in the illustration.

Install in this Direction

(b) Install the front seat inner belt assembly with the nut.

Torque:

42 N·m {428 kgf·cm, 31 ft·lbf}

NOTICE:

Do not allow the anchor part of the front seat inner belt assembly to overlap the protruding part of the separate type front seat cushion spring assembly.

(1) Engage the clamp.

*a

Protruding Part

(c) Engage each clamp.

(d) Connect each connector.
