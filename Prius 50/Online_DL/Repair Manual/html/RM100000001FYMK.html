Title: SUPPLEMENTAL RESTRAINT SYSTEMS: AIRBAG SYSTEM: B166A; Pressure Sensor Difference for Front Door; 2019 MY Prius [12/2018 - 08/2019]

DTC

B166A

Pressure Sensor Difference for Front Door

DESCRIPTION

The door side airbag sensor LH and door side airbag sensor RH detect impacts to the vehicle through changes in pressure inside the front doors caused by deformation of the outer side of the front doors and send signals to the airbag ECU assembly.

Based on the signals sent by the door side airbag sensors, the airbag ECU assembly determines if the airbags and pretensioners should be deployed.

If the difference between the pressure detected by the door side airbag sensor LH and the pressure detected by the door side airbag sensor RH exceeds a certain value a certain number of times, the airbag ECU assembly will store DTC B166A.

DTC No.

Detection Item

DTC Detection Condition

Trouble Area

Test Mode / Check Mode

B166A

Pressure Sensor Difference for Front Door

The difference between the pressure detected by the door side airbag sensor LH and the door side airbag sensor RH exceeds a certain value a certain number of times.

  • Door side airbag sensor LH
  • Door side airbag sensor RH
  • Airbag ECU assembly

Does not apply to test/check mode
