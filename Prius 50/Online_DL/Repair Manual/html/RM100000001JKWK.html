INSTALLATION

PROCEDURE

1. INSTALL NAVIGATION RECEIVER ASSEMBLY

2. INSTALL STEREO COMPONENT TUNER ASSEMBLY WITH WIRE (w/ Satellite Radio)

3. INSTALL NO. 2 RADIO BRACKET

(a) w/o Satellite Radio:

(1) Install the No. 2 radio bracket with the 3 screws.

(b) w/ Satellite Radio:

(1) Install the No. 2 radio bracket with the 5 screws.

4. INSTALL NO. 1 RADIO BRACKET

(a) w/o Satellite Radio:

(1) Install the No. 1 radio bracket with the 3 screws.

(b) w/ Satellite Radio:

(1) Install the No. 1 radio bracket with the 5 screws.

5. INSTALL HAZARD WARNING SIGNAL SWITCH ASSEMBLY

(a) Engage the 2 claws to install the hazard warning signal switch assembly.

6. CONNECT STEREO COMPONENT TUNER ASSEMBLY WITH WIRE (w/ Satellite Radio)

(a) Connect the 3 connectors to connect the stereo component tuner assembly with wire.

(b) Engage the clamp.

7. INSTALL CENTER INSTRUMENT CLUSTER FINISH PANEL SUB-ASSEMBLY

(a) Engage the 7 guides as shown in the illustration.

Install in this Direction

(b) Install the center instrument cluster finish panel sub-assembly with the 3 screws.

(a) Engage the 7 guides as shown in the illustration.

Install in this Direction

(b) Install the instrument cluster finish panel sub-assembly with the 3 screws.

(c) Connect the connector.
