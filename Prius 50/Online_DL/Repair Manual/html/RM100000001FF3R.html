Title: LIGHTING (EXT): AUTOMATIC HEADLIGHT BEAM LEVEL CONTROL SYSTEM: B2416,B241A; Height Control Sensor; 2019 - 2022 MY Prius [12/2018 -        ]

DTC

B2416

Height Control Sensor

DTC

B241A

Rear Height Control Sensor

DESCRIPTION

This DTC is stored when the No. 1 headlight ECU sub-assembly LH detects a malfunction in the rear height control sensor sub-assembly LH power source circuit or rear height control sensor sub-assembly LH circuit. The No. 1 headlight ECU sub-assembly LH stores DTC B2416 and B241A.

B2416

Height Control Sensor

  • Open or short in rear height control sensor sub-assembly LH power source circuit
  • Rear height control sensor sub-assembly LH
  • No. 1 headlight ECU sub-assembly LH
  • Harness or connector

Rear Height Control Sensor

(d) Enter the following menus: Body Electrical / AFS / Trouble Codes.

(e) Clear the DTCs.
