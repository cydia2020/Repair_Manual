Power Source Mode does not Change to ON (READY)

DESCRIPTION

When the electrical key transmitter sub-assembly is in the cabin and the power switch is pressed, the certification ECU (smart key ECU assembly) receives a signal and changes the power source mode. Additionally, when park (P) selected and the brake pedal is depressed, the hybrid control system can be started by pressing the power switch. If the gearshift control is unlocked, the hybrid control system can also be started by pressing the power switch with park (P) selected and the brake pedal depressed.
