INSTALLATION

CAUTION / NOTICE / HINT

HINT:

  • Use the same procedure for the RH side and LH side.
  • The following procedure is for the LH side.

PROCEDURE

1. INSPECT REAR 3 POINT TYPE SEAT OUTER BELT ASSEMBLY

Click here

2. INSTALL REAR 3 POINT TYPE SEAT OUTER BELT ASSEMBLY LH (for LH Side)

(a) Engage the 3 guides and temporarily install the rear 3 point type seat outer belt assembly LH with the 3 bolts.

Bolt (A)

Bolt (B)

(b) Uniformly tighten the 3 bolts in the order shown in the illustration to install rear 3 point type seat outer belt assembly LH.

Torque:

Bolt (A) :

12.5 N·m {127 kgf·cm, 9 ft·lbf}

Bolt (B) :

42 N·m {428 kgf·cm, 31 ft·lbf}

(c) Check that the ELR locks.

NOTICE:

This check should be performed with the rear 3 point type seat outer belt assembly LH installed to the vehicle.

(1) With the rear 3 point type seat outer belt assembly LH installed to the vehicle, check that the belt locks when it is pulled out quickly.

(d) Install the floor anchor of the rear 3 point type seat outer belt assembly LH with the bolt.

Torque:

42 N·m {428 kgf·cm, 31 ft·lbf}

NOTICE:

Do not allow the anchor part of the rear 3 point type seat outer belt assembly LH to overlap the protruding parts of the floor panel.

*a

Protruding Part
