INSTALLATION

PROCEDURE

1. INSTALL WINDSHIELD WASHER MOTOR AND PUMP ASSEMBLY

(a) w/o Rear Wiper:

(1) Install the washer motor and pump assembly as shown in the illustration.

Install in this Direction

(2) Connect the connector.

(b) w/ Rear Wiper:

(1) Install the washer motor and pump assembly as shown in the illustration.

Install in this Direction

(2) Connect the connector.

(3) Connect the washer hose.

2. ADD WASHER FLUID

(a) w/o Rear Wiper:

(1) Connect the washer hose.

(b) w/ Rear Wiper:

(1) Connect the rear washer hose.

(c) Add washer fluid to the washer jar.

3. INSTALL FRONT FENDER LINER RH

(a) Temporarily install the front fender liner RH with the 9 clips.

(b) for Rough Road Area Specification Vehicles:

(1) Install the clip.

(c) Install the 2 pin hold clips as shown in the illustration.

NOTICE:

Insert each pin hold clip with the slot aligned vertically. Do not rotate the pin hold clip after inserting it. After installation, confirm that the slot is aligned vertically.
