Title: LIGHTING (EXT): AUTOMATIC HIGH BEAM SYSTEM: B124B; Automatic High Beam System; 2021 - 2022 MY Prius [07/2020 -        ]

DTC

B124B

Automatic High Beam System

DESCRIPTION

This DTC is stored when the main body ECU (multiplex network body ECU) detects malfunctions in the automatic high beam system.

DTC No.

Detection Item

DTC Detection Condition

Trouble Area

B124B

Automatic High Beam System

Malfunction in automatic high beam system

(Detected for 2 seconds or more)

  • Front camera system
  • Dynamic radar cruise control system
  • Lane control system
  • Road sign assist system*
  • Front radar sensor system
  • Pre-collision system
  • Hybrid control system
  • Electronically controlled brake system
  • CAN communication system
  • Forward recognition camera
  • Main body ECU (multiplex network body ECU)
  • *: w/ Road Sign Assist System

HINT:

If the forward recognition camera stores a DTC for a malfunction of any of the following, this DTC will be stored simultaneously.

  • Speed sensor malfunction
  • Yaw rate sensor malfunction
  • Millimeter wave radar sensor malfunction
  • CAN communication system
