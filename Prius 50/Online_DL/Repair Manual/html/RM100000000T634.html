INSTALLATION

PROCEDURE

1. INSTALL STEREO COMPONENT TUNER ASSEMBLY WITH WIRE (w/ Satellite Radio)

2. INSTALL NAVIGATION RECEIVER ASSEMBLY

3. INSTALL NO. 2 RADIO BRACKET

(a) w/o Satellite Radio:

(1) Install the No. 2 radio bracket with the 3 screws.

(b) w/ Satellite Radio:

(1) Install the No. 2 radio bracket with the 5 screws.

4. INSTALL NO. 1 RADIO BRACKET

(a) w/o Satellite Radio:

(1) Install the No. 1 radio bracket with the 3 screws.

(b) w/ Satellite Radio:

(1) Install the No. 1 radio bracket with the 5 screws.

5. CONNECT STEREO COMPONENT TUNER ASSEMBLY WITH WIRE (w/ Satellite Radio)

(a) Connect the 3 connectors to connect the stereo component tuner assembly with wire.

6. INSTALL NAVIGATION RECEIVER ASSEMBLY WITH BRACKET

(a) Connect each connector.

(b) Engage the 2 guides to temporarily install the navigation receiver assembly with bracket as shown in the illustration.

Install in this Direction

(c) Install the navigation receiver assembly with bracket with the 4 bolts.
