DTC

U023587

Lost Communication with Cruise Control Front Distance Range Sensor Single Sensor or Center Missing Message

DESCRIPTION

The forward recognition camera communicates with the millimeter wave radar sensor assembly via CAN communication. If a communication malfunction between the forward recognition camera and millimeter wave radar sensor assembly is detected, DTC U023587 is stored.
