Parking Brake Switch Circuit

DESCRIPTION

The main body ECU (multiplex network body ECU) detects the condition of the parking brake switch assembly.

WIRING DIAGRAM

CAUTION / NOTICE / HINT

NOTICE:

Before replacing the main body ECU (multiplex network body ECU), refer to Registration.
