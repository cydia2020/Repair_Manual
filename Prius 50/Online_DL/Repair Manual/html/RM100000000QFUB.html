INSTALLATION

PROCEDURE

1. INSTALL NO. 1 AMPLIFIER ANTENNA ASSEMBLY

(a) Engage the guide.

*A

for Type A

*B

for Type B

(b) Install the No. 1 amplifier antenna assembly with the bolt.

Torque:

for Type A :

8.3 N·m {85 kgf·cm, 73 in·lbf}

for Type B :

10 N·m {102 kgf·cm, 7 ft·lbf}

NOTICE:

There are four types of bolts and the tightening torque depends on the type of bolt used as shown in the illustration. Therefore, confirm the tightening torque before installing the bolt.

(c) Connect the 2 connectors.

(d) Engage the claw.
