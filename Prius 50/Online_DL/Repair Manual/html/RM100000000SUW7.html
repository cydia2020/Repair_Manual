PROCEDURE

1. INSTALL HYBRID VEHICLE CONTROL ECU

(a) Engage the 2 claws to install the hybrid vehicle control ECU as shown in the illustration.

NOTICE:

Make sure that the hybrid vehicle control ECU is securely installed.

2. INSTALL ECU INTEGRATION BOX RH

(a) Install the ECU integration box RH with the bolt and 2 nuts.

(b) Engage the 3 clamps.

(c) Connect the 3 hybrid vehicle control ECU connectors and move each lock lever as shown in the illustration to lock the connectors.
