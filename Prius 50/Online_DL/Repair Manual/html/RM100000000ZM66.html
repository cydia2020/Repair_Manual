Title: METER / GAUGE / DISPLAY: METER / GAUGE SYSTEM: B150A; Lost Communication with HMI-LAN; 2017 - 2019 MY Prius [08/2016 - 08/2019]

DTC

B150A

Lost Communication with HMI-LAN

DESCRIPTION

The combination meter assembly (meter circuit plate) receives text data signals from the navigation receiver assembly via local bus communication.

Based on these signals, audio and visual system or navigation system information is displayed on the multi-information display.

This DTC is stored when the combination meter assembly (meter circuit plate) cannot receive the signal.

Note

B150A

Lost Communication with HMI-LAN

Diagnosis Condition:

  • A registration information signal was received from the Navigation receiver assembly when ACC power source voltage was received by the Navigation receiver assembly.
  • The power switch is on (IG).
  • IG power source voltage is 9.5 V or more.

Malfunction Status:

  • The registration information signal is not received from the Navigation receiver assembly.

Malfunction Time:

  • 30 seconds or more
  • Harness or connector
  • Navigation receiver assembly
  • Combination meter assembly (Meter circuit plate)
