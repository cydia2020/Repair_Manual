Title: 2ZR-FXE (BATTERY / CHARGING): AUXILIARY BATTERY: INSTALLATION; 2016 - 2018 MY Prius [11/2015 - 12/2018]

INSTALLATION

PROCEDURE

1. INSTALL AUXILIARY BATTERY

(a) w/ Battery Insulator:

(1) Install the battery insulator to the auxiliary battery.

(b) Install the auxiliary battery.

(c) Set the battery clamp sub-assembly as shown in the illustration.

Install in this Direction

(d) Install the bolt.

Torque:

15.4 N·m {157 kgf·cm, 11 ft·lbf}

(e) Connect the cable to the positive (+) auxiliary battery terminal and tighten the nut.

Torque:

5.6 N·m {57 kgf·cm, 50 in·lbf}

(f) Engage the 2 claws to close the battery terminal cap.

2. CONNECT CABLE TO NEGATIVE AUXILIARY BATTERY TERMINAL

(a) Connect the cable to the negative (-) auxiliary battery terminal and tighten the nut.

Torque:

5.4 N·m {55 kgf·cm, 48 in·lbf}

NOTICE:

When disconnecting the cable, some systems need to be initialized after the cable is reconnected.
