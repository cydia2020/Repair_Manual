Title: MIRROR (INT): INNER REAR VIEW MIRROR (w/ EC Mirror): INSTALLATION; 2016 - 2018 MY Prius [11/2015 - 12/2018]

INSTALLATION

PROCEDURE

1. INSTALL INNER REAR VIEW MIRROR ASSEMBLY (w/o Toyota Safety Sense P)

(a) Slide and temporarily install the inner rear view mirror assembly as shown in the illustration.

Install in this Direction

(b) Using a T20 "TORX" socket wrench, install the inner rear view mirror assembly with the screw.

Torque:

1.8 Nm {18 kgfcm, 16 inlbf}

(c) Connect the connector.

2. INSTALL INNER REAR VIEW MIRROR STAY HOLDER COVER (w/o Toyota Safety Sense P)

(a) Engage the 6 claws to install the inner rear view mirror stay holder cover as shown in the illustration.

Install in this Direction

(b) Slide the inner rear view mirror stay holder cover as shown in the illustration and engage the 2 guides.

Slide

3. INSTALL INNER REAR VIEW MIRROR ASSEMBLY (w/ Toyota Safety Sense P)

(a) Slide and temporarily install the inner rear view mirror assembly as shown in the illustration.

Install in this Direction

(b) Using a T20 "TORX" socket wrench, install the inner rear view mirror assembly with the screw.

Torque:

1.8 Nm {18 kgfcm, 16 inlbf}

(c) Engage the clamp.

(d) Connect the connector.
