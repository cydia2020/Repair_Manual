Title: LIGHTING (EXT): BACK-UP LIGHT BULB: REPLACEMENT; 2016 - 2018 MY Prius [11/2015 - 12/2018]

REPLACEMENT

CAUTION / NOTICE / HINT

HINT:

  • Use the same procedure for the RH side and LH side.
  • The following procedure is for the LH side.

PROCEDURE

1. REMOVE REAR COMBINATION LIGHT COVER

Click here

2. SEPARATE REAR COMBINATION LIGHT ASSEMBLY

(a) Apply protective tape around the rear combination light assembly as shown in the illustration.

Protective Tape

(b) Remove the 2 screws.

(c) Pull the rear combination light assembly toward the rear of the vehicle as shown in the illustration to disengage the 2 pins and 2 guides and separate the rear combination light assembly.

NOTICE:

To prevent the rear combination light assembly from falling when disengaging the pins and guides, lightly hold the rear combination light assembly.

*a

Pin

*b

Guide

Remove in this Direction

3. REMOVE BACK-UP LIGHT BULB

(a) Turn the rear combination light socket and wire sub-assembly with the back-up light bulb as shown in the illustration to disconnect them as a unit.

Remove in this Direction

(b) Remove the back-up light bulb from the rear combination light socket and wire sub-assembly.

4. INSTALL BACK-UP LIGHT BULB

(a) Install the back-up light bulb to the rear combination light socket and wire sub-assembly.

(b) Turn the rear combination light socket and wire sub-assembly with the back-up light bulb as shown in the illustration to connect them as a unit.

Install in this Direction

5. INSTALL REAR COMBINATION LIGHT ASSEMBLY

(a) Engage the 2 guides and 2 pins to temporarily install the rear combination light assembly as shown in the illustration.

*a

Pin

*b

Guide

Install in this Direction

(b) Install the rear combination light assembly with the 2 screws.

(c) Remove the protective tape.
