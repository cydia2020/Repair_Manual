Title: Q510 (HYBRID TRANSMISSION / TRANSAXLE): REAR TRACTION MOTOR: INSTALLATION; 2019 - 2022 MY Prius [12/2018 -        ]

INSTALLATION

PROCEDURE

1. INSTALL WIRE HARNESS CLAMP BRACKET

(a) Install the wire harness clamp bracket to the rear traction motor with transaxle assembly with the bolt.

Torque:

8.0 N·m {82 kgf·cm, 71 in·lbf}

2. INSTALL MOTOR CABLE BRACKET

(a) Install the motor cable bracket to the rear traction motor with transaxle assembly with the bolt.

Torque:

12 N·m {122 kgf·cm, 9 ft·lbf}

3. TEMPORARILY INSTALL FRONT DIFFERENTIAL SUPPORT ASSEMBLY

(a) Temporarily install the front differential support assembly to the rear traction motor with transaxle assembly with 2 new bolts.

4. INSTALL REAR TRACTION MOTOR WITH TRANSAXLE ASSEMBLY

(a) Using a mini crane and 2 belt slings, hold the rear traction motor at the position shown in the illustration.

(c) Fully tighten the 2 bolts (A).

(d) Fully tighten the 2 bolts (B).

Torque:

86 N·m {877 kgf·cm, 63 ft·lbf}

5. INSTALL MOTOR CABLE BRACKET

(a) Install the motor cable bracket to the rear suspension member sub-assembly with the bolt.

Torque:

12 N·m {122 kgf·cm, 9 ft·lbf}

6. CONNECT NO. 6 FLOOR WIRE

(a) Engage the 2 clamps.

(b) Connect the motor resolver sensor (motor temperature sensor) connector.
