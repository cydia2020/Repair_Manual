Title: LIGHTING (EXT): REAR COMBINATION LIGHT ASSEMBLY: INSTALLATION; 2019 - 2020 MY Prius [12/2018 - 07/2020]

INSTALLATION

CAUTION / NOTICE / HINT

HINT:

  • Use the same procedure for the RH side and LH side.
  • The following procedure is for the LH side.

PROCEDURE

1. INSTALL NO. 2 REAR COMBINATION LIGHT ASSEMBLY

(a) Engage the 3 claws to temporarily install the No. 2 rear combination light assembly.

(b) Install the No. 2 rear combination light assembly with the 3 screws.

(c) Connect the 2 connectors.
