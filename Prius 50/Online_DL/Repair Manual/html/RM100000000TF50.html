Title: HYBRID / BATTERY CONTROL: BATTERY ECU (for LITHIUM-ION BATTERY): INSTALLATION; 2016 - 2022 MY Prius [11/2015 -        ]

INSTALLATION

PROCEDURE

1. INSTALL NO. 2 HV BATTERY PROTECTOR

CAUTION:

Be sure to wear insulated gloves and protective goggles.

HINT:

Perform this procedure only when replacement of the No. 2 HV battery protector is necessary.

(a) Engage the claw and install the No. 2 HV battery protector to the HV battery.

(b) Connect each wire harness to the No. 2 HV battery protector.

2. INSTALL BATTERY ECU ASSEMBLY

CAUTION:

Be sure to wear insulated gloves and protective goggles.

(a) Install the battery ECU assembly to the HV battery with the 3 nuts.

Torque:

7.5 N·m {76 kgf·cm, 66 in·lbf}

NOTICE:

  • Check color of the label.

    *a

    Black Label

  • If the battery ECU assembly has been struck or dropped, replace it.

(b) Connect the 6 battery ECU assembly connectors.

NOTICE:

Make sure that the connectors are connected securely.

3. INSTALL NO. 1 HV BATTERY PROTECTOR

CAUTION:

Be sure to wear insulated gloves and protective goggles.

(a) Engage the 2 claws to install the No. 1 HV battery protector to the No. 2 HV battery protector.

(b) Install the 3 nuts.

Torque:

7.5 N·m {76 kgf·cm, 66 in·lbf}

(c) Engage the clamp to connect the wire harness.

4. INSTALL NO. 1 HYBRID BATTERY SHIELD SUB-ASSEMBLY

CAUTION:

Be sure to wear insulated gloves and protective goggles.

(a) Install the No. 1 hybrid battery shield sub-assembly to the HV battery with the nut.

Torque:

7.5 N·m {76 kgf·cm, 66 in·lbf}

(b) Engage the clamp.

5. INSTALL NO. 1 HV BATTERY HOSE

CAUTION:

Be sure to wear insulated gloves and protective goggles.

(a) Install the No. 1 HV battery hose to the HV battery with the clip.

6. INSTALL UPPER NO. 1 HYBRID BATTERY COVER SUB-ASSEMBLY

CAUTION:

Be sure to wear insulated gloves and protective goggles.

(a) Install the upper No. 1 hybrid battery cover sub-assembly to the HV battery with the 6 bolts and 5 nuts.

Torque:

7.5 N·m {76 kgf·cm, 66 in·lbf}
