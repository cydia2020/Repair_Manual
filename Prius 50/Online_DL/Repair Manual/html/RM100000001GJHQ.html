Title: TIRE PRESSURE MONITORING: TIRE PRESSURE WARNING SYSTEM: C2171; Transmitter ID not Registered (Main); 2019 - 2022 MY Prius [12/2018 -        ]

DTC

C2171

Transmitter ID not Registered (Main)

DESCRIPTION

The IDs of each tire pressure warning valve and transmitter are registered to the tire pressure warning ECU and receiver.

When the ECU detects that a transmitter ID code is not registered in the ECU, this DTC is stored.

DTC No.

Detection Item

DTC Detection Condition

Trouble Area

Note

C2171

Transmitter ID not Registered (Main)

Transmitter ID code is not registered. (When an ID code is unregistered for 3 minutes or more)

Tire pressure warning ECU and receiver

-

NOTICE:

ID Switching Function:

Even when the 2nd tire set is selected, "(Main)" is displayed on the Techstream.

CAUTION / NOTICE / HINT

NOTICE:

  • When replacing the tire pressure warning ECU and receiver, read the transmitter IDs and number of the transmitters (4 or 5) stored in the old ECU using the Techstream and write them down before removal.
  • It is necessary to perform initialization after registration of the transmitter IDs into the tire pressure warning ECU and receiver if the ECU has been replaced.
