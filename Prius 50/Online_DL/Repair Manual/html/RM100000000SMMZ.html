Title: LIGHTING (EXT): LIGHTING SYSTEM (w/o AFS): Daytime Running Light Relay Circuit; 2016 - 2020 MY Prius [11/2015 - 07/2020]

Daytime Running Light Relay Circuit

DESCRIPTION

The main body ECU (multiplex network body ECU) controls the daytime running lights.

WIRING DIAGRAM

CAUTION / NOTICE / HINT

NOTICE:

  • Inspect the fuses for circuits related to this system before performing the following procedure.
  • Before replacing the main body ECU (multiplex network body ECU), refer to Registration.

    Click here

1.
