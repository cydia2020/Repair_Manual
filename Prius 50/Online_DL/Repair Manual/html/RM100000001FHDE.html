FREEZE FRAME DATA

FREEZE FRAME DATA/INFORMATION

(a) Using the Techstream, check the vehicle condition (ECU, sensor) when the brake system operates or a DTC is output.

(b) The Freeze Frame Data that is recorded when a DTC is stored can be used to check the INF codes, but only for DTCs which can be stored.

CHECK FREEZE FRAME DATA WHEN BRAKE SYSTEM OPERATES

(a) Freeze Frame Data is stored when the brake system operates. The Freeze Frame Data is updated each time the system operates.

(b) Once Freeze Frame Data is stored when the system operates, it is not cleared until the DTC is cleared.

(c) The number of times (maximum: 31) the power switch has been turned from off to on (IG) since the last time the system was activated can be stored and displayed.

HINT:

  • When the vehicle is stopped (vehicle speed below 7 km/h (4 mph)), turning the power switch from off to on (IG) does not cause the value to increase.
  • The value "31" is displayed when the power switch was turned on (IG) more than 31 times.

(d) Check Freeze Frame Data when the system operates.

(1) Turn the power switch off.

(2) Connect the Techstream to the DLC3 and turn the power switch on (IG).

(3) Enter the following menus: Chassis / ABS/VSC/TRAC / Utility.

(4) Select "ABS History".

Chassis > ABS/VSC/TRAC > Utility

Tester Display

ABS History

(5) Check the Freeze Frame Data when the system operates.

CHECK FREEZE FRAME DATA AND INFORMATION WHEN DTC WAS STORED

(a) Freeze Frame Data is stored when a DTC is output.

(b) Once Freeze Frame Data is stored when a DTC is output, it is not updated or cleared until the DTC is cleared.

(c) The number of times (maximum: 31) the power switch has been turned from off to on (IG) since a DTC was output can be stored and displayed.

HINT:

The value "31" is displayed when the power switch was turned on (IG) more than 31 times.

(d) Check the Freeze Frame Data and INF code when a DTC is output.

(1) Turn the power switch off.

(2) Connect the Techstream to the DLC3 and turn the power switch on (IG).

(3) Enter the following menus: Chassis / ABS/VSC/TRAC / Trouble Codes.

Chassis > ABS/VSC/TRAC > Trouble Codes

(4) Select a DTC to display the Freeze Frame Data.

(5) Check the Freeze Frame Data for the output DTC.

HINT:

If an "INF Code" item appears when the Freeze Frame Data is displayed, select and check the INF code.

Chassis > ABS/VSC/TRAC

CLEAR FREEZE FRAME DATA AND INFORMATION

NOTICE:

Clearing the DTCs will also clear the Freeze Frame Data and information codes (INF codes).

(a) Turn the power switch off.

(b) Connect the Techstream to the DLC3.

(c) Turn the power switch on (IG).

(d) Turn the Techstream on.

(e) Clear the DTCs and Freeze Frame Data.

Click here

Chassis > ABS/VSC/TRAC > Clear DTCs
