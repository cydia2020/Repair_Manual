"P" Position Switch Indicator Light Circuit

DESCRIPTION

The P position switch (transmission shift main switch) indicator light comes on when the P position switch (transmission shift main switch) is pressed (park (P) is selected), goes off when a shift state other than park (P) is selected and blinks when a system malfunction occurs. A warning message will also be displayed on the multi-information display if a malfunction occurs.
