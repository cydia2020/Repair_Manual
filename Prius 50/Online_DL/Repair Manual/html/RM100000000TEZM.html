Title: METER / GAUGE / DISPLAY: COMBINATION METER: REASSEMBLY; 2016 - 2022 MY Prius [11/2015 -        ]

REASSEMBLY

PROCEDURE

1. INSTALL METER CIRCUIT PLATE

(a) Connect each connector as shown in the illustration.

Install in this Direction

NOTICE:

Be sure to connect the connector securely.

(b) Push down the 2 holders to engage the 4 claws as shown in the illustration.

NOTICE:

Be sure to engage the claws securely.

(c) Engage the 4 guides and 4 claws to install the meter circuit plate.

2. INSTALL NO. 2 METER CIRCUIT PLATE

(a) Engage the 4 guides as shown in the illustration.

Install in this Direction

(b) Install the No. 2 meter circuit plate with the 4 screws.

3. INSTALL NO. 2 COMBINATION METER CASE

(a) Engage the 2 guides to install the combination meter multi-display to the No. 2 combination meter case as shown in the illustration.

Install in this Direction

(b) Engage the 2 guides and connect the connector to install the No. 2 combination meter case with combination meter multi-display as shown in the illustration.

Install in this Direction

NOTICE:

  • Make sure to push the connector straight down when connecting it as it is easily damaged.
  • Make sure not to push the combination meter multi-display forcibly, or the connector may be damaged.
  • Be sure to connect the connector securely.

(c) Install the 2 screws.

(d) Engage the 2 guides.

(e) Install the No. 2 combination meter case with the screw.

4. INSTALL NO. 1 COMBINATION METER PLATE

(a) Engage the 3 guides to install the No. 1 combination meter plate.

5. INSTALL NO. 1 COMBINATION METER COVER

(a) Engage the 10 claws.

(b) Install the No. 1 combination meter cover with the 2 screws.

6. INSTALL COMBINATION METER GLASS

(a) Engage the 12 claws to install the combination meter glass.

(b) Install the 9 bushes.
