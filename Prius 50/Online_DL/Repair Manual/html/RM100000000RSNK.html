Title: LIGHTING (EXT): FRONT TURN SIGNAL LIGHT BULB: REPLACEMENT; 2016 - 2018 MY Prius [11/2015 - 12/2018]

REPLACEMENT

CAUTION / NOTICE / HINT

HINT:

  • Use the same procedure for the RH side and LH side.
  • The following procedure is for the LH side.

PROCEDURE

1. REMOVE FRONT TURN SIGNAL LIGHT BULB

(a) for RH Side:

Remove in this Direction

(1) Remove the clip and washer pipe as shown in the illustration.

(b) Turn the headlight cord with the front turn signal light bulb as shown in the illustration to disconnect them as a unit.

Remove in this Direction

(c) Remove the front turn signal light bulb from the headlight cord.

2. INSTALL FRONT TURN SIGNAL LIGHT BULB

(a) Install a new front turn signal light bulb to the headlight cord.

(b) Turn the headlight cord with the front turn signal light bulb as shown in the illustration to connect them as a unit.

Install in this Direction

(c) for RH Side:

(1) Install the washer pipe with the clip as shown in the illustration.

Install in this Direction
