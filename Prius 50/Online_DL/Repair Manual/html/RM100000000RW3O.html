Back-up Power Source Circuit

DESCRIPTION

The back-up power source circuit for the air conditioning amplifier assembly is shown below. Power is supplied even when the power switch is off. This power is used for diagnostic trouble code memory, etc.

CAUTION / NOTICE / HINT

NOTICE:

Inspect the fuses for circuits related to this system before performing the following procedure.
