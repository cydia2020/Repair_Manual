Title: STEERING GEAR / LINKAGE: STEERING GEAR: DISASSEMBLY; 2016 - 2022 MY Prius [11/2015 -        ]

DISASSEMBLY

PROCEDURE

1. REMOVE STEERING RACK BOOT CLIP (for LH Side)

(a) Using pliers, remove the steering rack boot clip.

2. REMOVE STEERING RACK BOOT CLIP (for RH Side)

HINT:

Perform the same procedure as for the LH side.

3. REMOVE STEERING RACK BOOT RETAINER (for LH Side)

(a) Using a screwdriver, remove the steering rack boot retainer.

NOTICE:

Be careful not to damage the No. 2 steering rack boot.

4. REMOVE STEERING RACK BOOT RETAINER (for RH Side)

HINT:

Perform the same procedure as for the LH side.

5. REMOVE NO. 2 STEERING RACK BOOT

6. REMOVE NO. 1 STEERING RACK BOOT
