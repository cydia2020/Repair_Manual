Title: REAR SUSPENSION: REAR UPPER ARM: INSTALLATION; 2019 - 2022 MY Prius [12/2018 -        ]

INSTALLATION

PROCEDURE

1. INSTALL REAR UPPER CONTROL ARM ASSEMBLY LH (for 2WD)

(a) Temporarily install the rear upper control arm assembly LH to the rear suspension member sub-assembly with the bolt and nut.

NOTICE:

  • Because the nut has its own stopper, do not turn the nut. Tighten the bolt with the nut secured.
  • Insert the bolt with the threaded end facing the front of the vehicle.

(b) Position the rear upper control arm assembly LH as shown in the illustration.

Reference Length (A):

1.2 mm (0.0472 in.)

(c) Fully tighten the bolt.

Torque:

73 N·m {744 kgf·cm, 54 ft·lbf}

NOTICE:

Because the nut has its own stopper, do not turn the nut. Tighten the bolt with the nut secured.

2. INSTALL REAR UPPER CONTROL ARM ASSEMBLY RH (for 2WD)

HINT:

Perform the same procedure as for the LH side.

3. INSTALL REAR UPPER CONTROL ARM ASSEMBLY LH (for AWD)

(a) Temporarily install the rear upper control arm assembly LH to the rear suspension member sub-assembly with the bolt and nut.
