Title: LIGHTING (EXT): CLEARANCE LIGHT BULB: REPLACEMENT; 2019 - 2022 MY Prius [12/2018 -        ]

REPLACEMENT

CAUTION / NOTICE / HINT

HINT:

  • Use the same procedure for the RH side and LH side.
  • The following procedure is for the LH side.

PROCEDURE

1. REMOVE CLEARANCE LIGHT BULB

(a) Turn the headlight cord with the clearance light bulb as shown in the illustration to disconnect them as a unit.

Remove in this Direction

(b) Remove the clearance light bulb from the headlight cord.

2. INSTALL CLEARANCE LIGHT BULB

(a) Install the clearance light bulb to the headlight cord.

(b) Turn the headlight cord with the clearance light bulb as shown in the illustration to connect them as a unit.

Install in this Direction
