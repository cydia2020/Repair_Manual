FAIL-SAFE CHART

FAIL-SAFE FUNCTION

(a) When communication fails in any of the CAN bus lines (communication lines), a fail-safe function(s) will operate. The fail-safe function that is specified for each system operates to prevent those systems from malfunctioning.

(b) The following table shows the effects on each system when communication is impossible. (For further details, see the pages for each system.)
