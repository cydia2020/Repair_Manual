Title: 2ZR-FXE (BATTERY / CHARGING): AUXILIARY BATTERY: INSTALLATION; 2019 - 2022 MY Prius [12/2018 -        ]

INSTALLATION

PROCEDURE

1. INSTALL AUXILIARY BATTERY

(a) Install the auxiliary battery.

(b) Set the battery clamp sub-assembly as shown in the illustration.

Install in this Direction

(c) Install the bolt.

Torque:

15.4 N·m {157 kgf·cm, 11 ft·lbf}

(d) Connect the cable to the positive (+) auxiliary battery terminal and tighten the nut.

Torque:

5.6 N·m {57 kgf·cm, 50 in·lbf}

(e) Engage the 2 claws to close the battery terminal cap.

2. CONNECT CABLE TO NEGATIVE AUXILIARY BATTERY TERMINAL

(a) Connect the cable to the negative (-) auxiliary battery terminal and tighten the nut.

Torque:

5.4 N·m {55 kgf·cm, 48 in·lbf}

NOTICE:

When disconnecting the cable, some systems need to be initialized after the cable is reconnected.
