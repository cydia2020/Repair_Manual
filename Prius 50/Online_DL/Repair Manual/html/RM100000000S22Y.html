DATA LIST / ACTIVE TEST

DATA LIST

NOTICE:

In the table below, the values listed under "Normal Condition" are reference values. Do not depend solely on these reference values when deciding whether a part is faulty or not.

HINT:

Using the Techstream to read the Data List allows the values or states of switches, sensors, actuators and other items to be read without removing any parts. This non-intrusive inspection can be very useful because intermittent conditions or signals may be discovered before parts or wiring is disturbed. Reading the Data List information early in troubleshooting is one way to save diagnostic time.

(a) Connect the Techstream to the DLC3.

(b) Turn the power switch on (IG).

(c) Turn the Techstream on.

(d) Enter the following menus: Body Electrical / Smart Key / Data List.

(e) Read the Data List according to the display on the Techstream.

Body Electrical > Smart Key > Data List

Unlock Request Receive

Status of gearshift control unlock command from certification ECU (smart key ECU assembly)*7,*9

OK or NG

OK: Certification ECU (smart key ECU assembly) sends gearshift control unlock command (within 10 seconds of power switch turned on (ACC) or on (IG), or of hybrid control system start operation performed)

NG: Certification ECU (smart key ECU assembly) does not send gearshift control unlock command (power switch not turned on (ACC) or on (IG), and hybrid control system start operation not performed)

If OK is not displayed even when the gearshift control unlock conditions are met, the certification ECU (smart key ECU assembly) may be malfunctioning.

Lock Request Receive

Status of gearshift control lock command from certification ECU (smart key ECU assembly)*8,*9

OK or NG

OK: Certification ECU (smart key ECU assembly) sends gearshift control lock command (within 10 seconds of any door opened with park (P) selected and power switch off)

NG: Certification ECU (smart key ECU assembly) does not send gearshift control lock command (no door opened with park (P) selected and power switch off)

If OK is not displayed even when the gearshift control lock conditions are met, the certification ECU (smart key ECU assembly) may be malfunctioning.

HINT:

  • *1: This indicates that there is a problem with the communication format between the electrical key transmitter sub-assembly and certification ECU (smart key ECU assembly). Wave interference, malfunction of the electrical key transmitter sub-assembly or certification ECU (smart key ECU assembly), an electrical key transmitter sub-assembly from different vehicle being used, etc. are possible causes.
  • *2: Hybrid control system start is permitted when the gearshift control is determined to be locked and is not malfunctioning. If hybrid control system start is not permitted, the hybrid vehicle control ECU may be malfunctioning.
  • *3: Transmission of the wake-up signal indicates that certification system related LIN communication is being performed. The value of this item is "Yes" when the gearshift control lock motor is operating or communication is being performed between the certification ECU (smart key ECU assembly) and hybrid vehicle control ECU.
  • *4: This indicates that the hybrid control system start permission request signal (EGST) output from the hybrid vehicle control ECU is being received by the ID code box (immobiliser code ECU).
  • *5: This indicates the certification result of the certification codes of the certification ECU (smart key ECU assembly) and ID code box (immobiliser code ECU).
  • *6: This indicates the certification result of the certification codes of the ID code box (immobiliser code ECU) and hybrid vehicle control ECU.

    When the hybrid vehicle control ECU receives an unlock request from the certification ECU (smart key ECU assembly), confirmation of the ID code box (immobiliser code ECU) L code is performed. If the L code matches, the hybrid vehicle control ECU unlocks the gearshift control. Also, if the S code verification result does not match, the gearshift control lock will not be unlocked.

  • *7: The certification ECU (smart key ECU assembly) outputs a gearshift control unlock request for 10 seconds after the power switch is turned on (ACC) or on (IG), or a hybrid control system start operation is performed.
  • *8: The certification ECU (smart key ECU assembly) outputs a gearshift control lock request for 10 seconds after the power switch is turned off with the park (P) selected.
  • *9: Refer to the following Data List items when inspecting the actual vehicle.

    With the power switch off and the park (P) selected, perform the following procedures and check the power source mode.

    • *1: Without depressing the brake pedal, press the power switch and check that the power source mode changes to on (ACC).
    • *2: While depressing the brake pedal, press the power switch and check that the hybrid control system starts.
    • *3: Press the power switch and check that the power source mode changes to off.
    • *4: Power source mode turns off.

ACTIVE TEST

HINT:

Using the Techstream to perform Active Tests allows relays, VSVs, actuators and other items to be operated without removing any parts. This non-intrusive functional inspection can be very useful because intermittent operation may be discovered before parts or wiring is disturbed. Performing Active Tests early in troubleshooting is one way to save diagnostic time. Data List information can be displayed while performing Active Tests.

(a) Connect the Techstream to the DLC3.

(b) Turn the power switch on (IG).

(c) Turn the Techstream on.

(d) Enter the following menus: Body Electrical / Smart Key / Active Test.

(e) Perform the Active Test according to the display on the Techstream.
