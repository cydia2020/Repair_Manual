4. REMOVE HOOD LOCK ASSEMBLY

(a) Using a screwdriver with its tip wrapped with protective tape, remove the hood lock nut cap.

6. DISCONNECT HOOD LOCK CONTROL LEVER SUB-ASSEMBLY

Click here

7. REMOVE HOOD LOCK CONTROL LEVER SUB-ASSEMBLY

(a) for Type A:

(1) Disconnect the hood lock control cable assembly to remove the hood lock control lever sub-assembly.

HINT:

When changing the hood lock control lever sub-assembly from type A to type B, the hood lock control cable assembly must also be changed to type B.

(b) for Type B:

(1) Disconnect the hood lock control cable assembly to remove the hood lock control lever sub-assembly.

HINT:

When changing the hood lock control lever sub-assembly from type B to type A, the hood lock control cable assembly must also be changed to type A.
