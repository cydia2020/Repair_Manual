Title: THEFT DETERRENT / KEYLESS ENTRY: SMART KEY SYSTEM (for Entry Function): OPERATION HISTORY LIST; 2016 - 2022 MY Prius [11/2015 -        ]

OPERATION HISTORY LIST

NOTICE:

  • The cause of a malfunction is stored in the RAM or EEPROM in the certification ECU (smart key ECU assembly). As the cause of a malfunction stored in the RAM will be cleared when the cable is disconnected from the negative (-) auxiliary battery terminal, do not disconnect the cable from the negative (-) auxiliary battery terminal before checking and recording the operation history.
  • When using the Techstream with the power switch off, connect the Techstream to the DLC3 and turn a courtesy light switch on and off at intervals of 1.5 seconds or less until communication between the Techstream and the vehicle begins. Then select the vehicle type under manual mode and enter the following menus: Body Electrical / Smart Key. While using the Techstream, periodically turn a courtesy light switch on and off at intervals of 1.5 seconds or less to maintain communication between the Techstream and the vehicle.

OPERATION HISTORY

HINT:

The certification ECU (smart key ECU assembly) stores the operation history of the smart key system and it can be read using the Techstream.

(a) Connect the Techstream to the DLC3.

(b) Turn the power switch on (IG).

(c) Turn the Techstream on.

(d) Enter the following menus: Body Electrical / Smart Key / Utility / Operation History / Certification ECU operation log of ID code registration, Radio wave interference or Radio wave interference 2.

Certification ECU operation log of ID code registration

Parameter Name

Content

Transponder Key Amp Error

Although an unregistered electrical key transmitter sub-assembly was held near the power switch, electrical key transmitter sub-assembly registration was not successful after one or more ID codes had successfully been registered due to a malfunction of the immobiliser amplifier, such as a short to ground. The ECU code was registered successfully. DTC B278A was stored.

Registration Failed; Vehicle ID Error

Although an unregistered electrical key transmitter sub-assembly was held near the power switch, electrical key transmitter sub-assembly registration was suspended after one or more ID codes had successfully been registered because time had expired due to the vehicle ID code sent by the electrical key transmitter sub-assembly not matching the code stored by the certification ECU (smart key ECU assembly).

A recognition code is used between the electrical key transmitter sub-assembly and the certification ECU (smart key ECU assembly). Although the vehicle ID stored in the certification ECU (smart key ECU assembly) should be registered to the electrical key transmitter sub-assembly when the electrical key transmitter sub-assembly was registered, it was not. "Registration Failed; Vehicle ID Error" is displayed when trying to register an electrical key transmitter sub-assembly of another vehicle. If electrical key transmitter sub-assembly registration could not be completed successfully after trying to register another electrical key transmitter sub-assembly, the certification ECU (smart key ECU assembly) might be malfunctioning.

Registration Failed; Except Vehicle ID Error

Although an unregistered electrical key transmitter sub-assembly was held near the power switch, electrical key transmitter sub-assembly registration was suspended after one or more ID codes had successfully been registered due to noise or the transmitter battery being low.

Registration Failed; Key Already Registered

As a registered electrical key transmitter sub-assembly was held near the power switch by mistake, electrical key transmitter sub-assembly registration was suspended after one or more ID codes had successfully been registered.

Registration Failed; ID Code Write Error

Although an unregistered electrical key transmitter sub-assembly was held near the power switch, electrical key transmitter sub-assembly registration was suspended after one or more ID codes had successfully been registered due to the certification ECU (smart key ECU assembly) not being able to store the electrical key transmitter sub-assembly information correctly.

Registration Failed; Key Info Abnormal

Although an unregistered electrical key transmitter sub-assembly was held near the power switch, electrical key transmitter sub-assembly registration was suspended after one or more ID codes had successfully been registered due to the certification ECU (smart key ECU assembly) not being able to store the electrical key transmitter sub-assembly ID code correctly. The ECU code was registered successfully.

The following is suspected:

Registration of electrical key transmitter sub-assemblies can only be performed one at a time. If 2 electrical key transmitter sub-assemblies were held near the power switch at the same time, incorrect registration information was stored in the certification ECU (smart key ECU assembly), suspending registration.

HINT:

Reregister the electrical key transmitter sub-assemblies using the correct procedure.

Failed Registration Mode

Although the mode was changed to registration mode using the Techstream, the mode was not successfully changed (another mode may have already been selected).

Vehicle ID not set

Although the vehicle ID should be stored in the certification ECU (smart key ECU assembly) at the factory, the ID could not be stored successfully (malfunction of the certification ECU (smart key ECU assembly)).

Key Registration Max

Registration was not possible because the maximum number of electrical key transmitter sub-assemblies had already been registered to the certification ECU (smart key ECU assembly).

HINT:

  • If a new electrical key transmitter sub-assembly needs to be registered, such as when one of the registered electrical key transmitter sub-assemblies has been lost, erase an unnecessary ID code first.
  • If the certification ECU (smart key ECU assembly) has been replaced with a new one, there should be no registered electrical key transmitter sub-assemblies. If any exist, replace the new certification ECU (smart key ECU assembly) with another new certification ECU (smart key ECU assembly).

ID-BOX Error

Entering registration mode was not possible because whether the vehicle was provided with an ID code box was unknown. (If the function of the ID code box is integrated into the certification ECU (smart key ECU assembly), no ID code box is provided. If it is not integrated, an ID code box is provided.)

Vehicle Speed Too High

Electrical key transmitter sub-assembly registration was not possible because the vehicle speed was 5 km/h or more (electrical key transmitter sub-assembly registration is possible only while the vehicle is stopped).

Transponder Key Amp Abnormal

Although a registered electrical key transmitter sub-assembly was held near the power switch, the registered electrical key transmitter sub-assemblies could not be confirmed and electrical key transmitter sub-assembly registration was not possible due to a malfunction of the immobiliser amplifier, such as a short to ground. DTC B278A was stored.

Immobiliser Verification Error

Although a registered electrical key transmitter sub-assembly was held near the power switch, ID number certification between the electrical key transmitter sub-assembly and the certification ECU (smart key ECU assembly) was not successful.

The following are suspected:

  • The registered electrical key transmitter sub-assembly was not held near the power switch properly.
  • Malfunction of the power switch (immobiliser amplifier), related wire harness or electrical key transmitter sub-assembly.

HINT:

The immobiliser system certification procedure is performed by holding an electrical key transmitter sub-assembly near the power switch and checking if whether the electrical key transmitter sub-assembly ID information matches that of the certification ECU (smart key ECU assembly).

No ECU Response

A recognition code is used between the certification ECU (smart key ECU assembly) and hybrid vehicle control ECU. Electrical key transmitter sub-assembly registration was not possible due to the certification ECU (smart key ECU assembly) not receiving a response from the hybrid vehicle control ECU. If no malfunction was found in the ECU connectors or wire harnesses during inspection, a malfunction in either of the ECUs is suspected.

S Code Verification Error

A recognition code (S code) is used between the certification ECU (smart key ECU assembly) and ID code box (immobiliser code ECU). Registration was not possible due to these codes not matching.

If no malfunction was found in all the ECU connectors or wire harnesses at inspection, a malfunction in one of the ECUs is suspected.

HINT:

If an ECU has been registered to another vehicle in the past the codes will not match.

L Code Verification Error

A recognition code (L code) is used between the ID code box (immobiliser code ECU) and hybrid vehicle control ECU. Registration was not possible due to these codes not matching.

If a malfunction was not found in any of the connectors or wire harnesses, a malfunction in one of the ECUs is suspected.

HINT:

If an ECU has been registered to another vehicle in the past the codes will not match.

L Code OK S Code NG

Although L code verification was successful, registration was not possible due to S code verification result being abnormal.

HINT:

A recognition code (S code) is used between the certification ECU (smart key ECU assembly) and ID code box (immobiliser code ECU). A recognition code (L code) is used between the ID code box (immobiliser code ECU) and hybrid vehicle control ECU. If an ECU has been registered to another vehicle in the past the codes will not match.

Key Verification: Max Number

The number of registered electrical key transmitter sub-assemblies stored in the certification ECU (smart key ECU assembly) exceeded the maximum.

Key Verification: Amp Abnormal

Although all of the electrical key transmitter sub-assemblies had been held near the power switch to confirm their registration, the registered electrical key transmitter sub-assemblies could not be confirmed and electrical key transmitter sub-assembly registration was not possible due to a malfunction of the immobiliser amplifier, such as a short to ground. DTC B278A was stored.

Key Verification: Read Error

Although all of the electrical key transmitter sub-assemblies had been held near the power switch to confirm their registration, electrical key transmitter sub-assembly registration was not possible due to immobiliser recognition not succeeding.

The following are suspected:

  • The registered electrical key transmitter sub-assembly was not held near the power switch properly.
  • Malfunction of the power switch (immobiliser amplifier), electrical key transmitter sub-assembly or a related wire harness.

HINT:

The immobiliser system certification procedure is performed by holding an electrical key transmitter sub-assembly near the power switch and checking if the electrical key transmitter sub-assembly ID information matches that of the certification ECU (smart key ECU assembly).

Unverified Key

Although all of the electrical key transmitter sub-assemblies had been held near the power switch to confirm their registration, electrical key transmitter sub-assembly registration was not possible due to some of the electrical key transmitter sub-assemblies being unconfirmed.

The following are suspected:

  • Some of the registered electrical key transmitter sub-assemblies were not held near the power switch properly.
  • Some of the registered electrical key transmitter sub-assemblies had not been held near the power switch yet.

HINT:

Prepare all of the registered electrical key transmitter sub-assemblies and restart system certification.

Key Info Abnormal

Although all of the electrical key transmitter sub-assemblies had been held near the power switch to confirm their registration, electrical key transmitter sub-assembly registration was not possible due to each key ID not being registered correctly in the certification ECU (smart key ECU assembly).

The following is suspected:

Registration of electrical key transmitter sub-assemblies can only be performed one at a time. If 2 electrical key transmitter sub-assemblies were held near the power switch at the same time, incorrect registration information was stored in the certification ECU (smart key ECU assembly), suspending registration.

HINT:

Register the electrical key transmitter sub-assemblies using the correct procedure.

Transponder Key Amp Error

Although an unregistered electrical key transmitter sub-assembly was held near the power switch, the electrical key transmitter sub-assembly registration was not successful after one or more ID code registration had been completed successfully due to a malfunction of the immobiliser amplifier (short to ground). The ECU code was registered successfully. DTC B278A was stored.

New Key Registration Error

Although an unregistered electrical key transmitter sub-assembly was held near the power switch, electrical key transmitter sub-assembly registration was not possible due to immobiliser system certification not succeeding.

The following are suspected:

  • The registered electrical key transmitter sub-assembly was not held near the power switch properly.
  • Malfunctions of the power switch (immobiliser amplifier), a related wire harness or an electrical key transmitter sub-assembly.

HINT:

The immobiliser system certification procedure is performed by holding an electrical key transmitter sub-assembly near the power switch and checking if whether or not the electrical key transmitter sub-assembly ID information matches that of the certification ECU (smart key ECU assembly).

New Key Vehicle ID Error

Although an unregistered electrical key transmitter sub-assembly was held near the power switch, electrical key transmitter sub-assembly registration was not possible due to the vehicle ID code sent by the electrical key transmitter sub-assembly not matching the code stored by the certification ECU (smart key ECU assembly).

A recognition code is used between the electrical key transmitter sub-assembly and the certification ECU (smart key ECU assembly). Although the vehicle ID stored in the certification ECU (smart key ECU assembly) should be registered to the electrical key transmitter sub-assembly on the occasion of the electrical key transmitter sub-assembly registration, it was not successful.

"Registration Failed; Vehicle ID Error" was displayed when trying to register an electrical key transmitter sub-assembly of another vehicle. If the electrical key transmitter sub-assembly registration could not be completed successfully after trying to register another electrical key transmitter sub-assembly, the certification ECU (smart key ECU assembly) may be malfunctioning.

New Key Registration Failed

Although an unregistered electrical key transmitter sub-assembly was held near the power switch, electrical key transmitter sub-assembly registration was not possible due to the key ID code sent by the electrical key transmitter sub-assembly not matching the code stored by the certification ECU (smart key ECU assembly) due to noise or the transmitter battery being low.

"New Key Vehicle ID Error" was displayed due to the vehicle ID registered in the certification ECU (smart key ECU assembly) and the vehicle ID registered in the electrical key transmitter sub-assembly not matching.

New Key Already Registered

Electrical key transmitter sub-assembly registration was not possible due to a registered electrical key transmitter sub-assembly being held near the power switch instead of an unregistered electrical key transmitter sub-assembly by mistake.

New Key ID Code Error

Although an unregistered electrical key transmitter sub-assembly was held near the power switch, electrical key transmitter sub-assembly registration was not possible due to the certification ECU (smart key ECU assembly) not being able to store the key information correctly. If registration cannot be completed successfully after a second attempt, the certification ECU (smart key ECU assembly) is malfunctioning.

New Key Information Abnormal

Although an unregistered electrical key transmitter sub-assembly was held near the power switch, electrical key transmitter sub-assembly registration was not possible due to the certification ECU (smart key ECU assembly) not being able to store the key information correctly.

The following is suspected:

Registration of electrical key transmitter sub-assemblies can only be performed one at a time. If 2 electrical key transmitter sub-assemblies were held near the power switch at the same time, incorrect registration information was stored in the certification ECU (smart key ECU assembly), suspending registration.

HINT:

Reregister the electrical key transmitter sub-assemblies using the correct procedure.

No Response Registering L/S Code

Registration of the hybrid vehicle control ECU was not possible due to the certification ECU (smart key ECU assembly) not receiving a response from the hybrid vehicle control ECU.

HINT:

Inspect the connectors and wire harnesses.

L Code Verification NG

L code registration was not possible. A recognition code (L code) is used between the ID code box (immobiliser code ECU) and the hybrid vehicle control ECU.

The following is suspected:

A malfunction occurred in a connector, wire harness or ECU.

S Code Verification NG

S code registration was not possible. A recognition code (S code) is used between the ID code box (immobiliser code ECU) and the certification ECU (smart key ECU assembly).

The following is suspected:

A malfunction occurred in a connector, wire harness or ECU.

ECU ID Verification Error

Although the ID code box (immobiliser code ECU) sent an S code registration request to the certification ECU (smart key ECU assembly), ECU code registration was not possible due to the input data being incorrect.

Key Number Write Error

ECU code registration was not possible due to the certification ECU (smart key ECU assembly) not being able to correctly store the number of registered electrical key transmitter sub-assemblies.

ID Code BOX Write Error

ECU code registration was not possible due to the certification ECU (smart key ECU assembly) not being able to store the information of whether an ID code box is provided.

B Code Verification NG

B code registration was not possible. A recognition code (B code) is used between the certification ECU (smart key ECU assembly) and the main body ECU (multiplex network body ECU).

The following is suspected:

A malfunction occurred in a connector, wire harness or ECU.

Radio wave interference

Parameter Name

Content

Lock / Key RF Signal Interference

When an entry lock operation was performed, the electrical key transmitter sub-assembly could not be confirmed due to wave interference.

Unlock / Key RF Signal Interference

When an entry unlock operation was performed, the electrical key transmitter sub-assembly could not be confirmed due to wave interference.

Operating Engine Switch / Key RF Signal Interference

When the power switch was operated, the electrical key transmitter sub-assembly could not be confirmed due to wave interference.

Operating Trunk Switch / Key RF Signal Interference

When an entry back door unlock operation was performed, the electrical key transmitter sub-assembly could not be confirmed due to wave interference.

Radio wave interference 2

Parameter Name

Content

Lock / Key RF Signal Interference

When an entry lock operation was performed, the electrical key transmitter sub-assembly could not be verified because it was excessively close to an electrical key antenna or because a tool or transmitter other than a registered electrical key transmitter sub-assembly was used.

Unlock / Key RF Signal Interference

When an entry unlock operation was performed, the electrical key transmitter sub-assembly could not be verified because it was excessively close to an electrical key antenna or because a tool or transmitter other than a registered electrical key transmitter sub-assembly was used.

Operating Engine Switch / Key RF Signal Interference

When the power switch was operated, the electrical key transmitter sub-assembly could not be verified because it was excessively close to an indoor electrical key antenna assembly or because a tool or transmitter other than a registered electrical key transmitter sub-assembly was used.

Operating Trunk Switch / Key RF Signal Interference

When an entry back door unlock operation was performed, the electrical key transmitter sub-assembly could not be verified because it was excessively close to the back door opener switch assembly (electrical key antenna) or because a tool or transmitter other than a registered electrical key transmitter sub-assembly was used.
