REASSEMBLY

CAUTION / NOTICE / HINT

HINT:

  • Use the same procedure for the RH side and LH side.
  • The following procedure is for the LH side.

PROCEDURE

1. INSTALL HEADLIGHT UNIT ASSEMBLY

2. INSTALL FRONT MARKER LIGHT SOCKET AND WIRE

3. INSTALL FRONT SIDE MARKER LIGHT BULB

(a) Install the front side marker light bulb to the front marker light socket and wire.

(b) Turn the front marker light socket and wire with the front side marker light bulb as shown in the illustration to install them as a unit.

Install in this Direction

(c) Connect the connector.

4. INSTALL HEADLIGHT FRONT TURN SIGNAL LED

(a) Turn the headlight front turn signal LED as shown in the illustration to install it.

Install in this Direction

(b) Connect the connector.

5. INSTALL CLEARANCE LIGHT SOCKET (for Bulb Type Clearance Light)

6. INSTALL CLEARANCE LIGHT BULB (for Bulb Type Clearance Light)

(a) Install the clearance light bulb to the clearance light socket.

(b) Turn the clearance light socket with the clearance light bulb as shown in the illustration to install them as a unit.

Install in this Direction

(c) Connect the connector.
