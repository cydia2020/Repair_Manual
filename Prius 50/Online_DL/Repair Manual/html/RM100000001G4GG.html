Turn Signal Switch Circuit

DESCRIPTION

The combination meter assembly (meter circuit plate) receives the turn signal switch information and controls the turn signal lights.

WIRING DIAGRAM

PROCEDURE

1.

READ VALUE USING TECHSTREAM

(a) Connect the Techstream to the DLC3.

(b) Turn the power switch on (IG).

(c) Turn the Techstream on.

(d) Enter the following menus: Body Electrical / Combination Meter / Data List.

(e) Read the Data List according to the display on the Techstream.
