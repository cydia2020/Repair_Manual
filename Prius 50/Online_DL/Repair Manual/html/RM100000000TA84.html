Title: LIGHTING (INT): LIGHTING SYSTEM: Footwell Light Circuit; 2016 - 2022 MY Prius [11/2015 -        ]

Footwell Light Circuit

DESCRIPTION

The main body ECU (multiplex network body ECU) controls the footwell lights.

WIRING DIAGRAM

CAUTION / NOTICE / HINT

NOTICE:

  • Inspect the fuses for circuits related to this system before performing the following inspection procedure.
  • Before replacing the main body ECU (multiplex network body ECU), refer to Registration.
