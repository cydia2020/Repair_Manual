DTC

B1532

LVDS Signal Malfunction (from Extension Module)

DESCRIPTION

The stereo component tuner assembly and the navigation receiver assembly are connected via LVDS communication line.

This DTC is stored when an LVDS communication error occurs between the stereo component tuner assembly and the navigation receiver assembly.

DTC No.

Detection Item

DTC Detection Condition

Trouble Area

B1532

LVDS Signal Malfunction (from Extension Module)

When any of the following conditions is met:

  • Video signal (digital) is lost.
  • Stereo component tuner assembly is/was not connected while the power switch is/was on (ACC) or (IG).
  • Communication between the navigation receiver assembly and the stereo component tuner assembly is/was not possible when the engine is/was started.
  • Stereo component tuner assembly power source circuit
  • LVDS circuit between navigation receiver assembly and stereo component tuner assembly
  • Stereo component tuner assembly
  • Navigation receiver assembly
  • Navigation wire

HINT:

Even if no malfunction is present, this DTC may be stored depending on the auxiliary battery condition or engine start voltage.

WIRING DIAGRAM

CAUTION / NOTICE / HINT
