Title: LIGHTING (EXT): LIGHTING SYSTEM (w/ AFS): B1244; Light Sensor Circuit; 2019 - 2022 MY Prius [12/2018 -        ]

DTC

B1244

Light Sensor Circuit

DESCRIPTION

The automatic light control sensor detects ambient light. The sensor creates an electrical signal based on the amount of light detected, and sends the signal to the main body ECU (multiplex network body ECU). The main body ECU (multiplex network body ECU) turns on or off the headlights and taillights according to the signal. The main body ECU (multiplex network body ECU) stores DTC B1244.

Trouble Area

Note

B1244

Light Sensor Circuit

Either of the following conditions is met:

  • Malfunction in automatic light control sensor
  • Open or short in automatic light control sensor circuit
  • Automatic light control sensor
  • Harness or connector
  • Main body ECU (multiplex network body ECU)
