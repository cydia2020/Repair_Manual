DTC

B1503

Exhaust Heat Management Warning Detected

DESCRIPTION

This DTC is stored when the combination meter assembly (meter circuit plate) detects a malfunction in the engine coolant temperature sensor that is connected to the combination meter assembly (meter circuit plate) via a direct line.

DTC No.

Detection Item

DTC Detection Condition

Trouble Area

Memory

Note

B1503

Exhaust Heat Management Warning Detected

When IG voltage is 9.5 V or more and either of the following conditions is detected:

  • Engine coolant temperature sensor circuit open or shorted
  • Engine coolant temperature is 120°C (248°F) or more
  • Harness or connector
  • Engine coolant temperature sensor
  • Combination meter assembly (Meter circuit plate)

DTC stored

w/ Exhaust Heat Recirculation System

HINT:

This DTC is for the engine coolant temperature sensor which is used to detect the engine coolant temperature for the exhaust heat recirculation system, not for the engine coolant temperature sensor which is used for the SFI system.
