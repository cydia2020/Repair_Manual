Title: METER / GAUGE / DISPLAY: HEADUP DISPLAY SYSTEM: B150A; Lost Communication with HMI-LAN; 2016 MY Prius [11/2015 - 08/2016]

DTC

B150A

Lost Communication with HMI-LAN

DESCRIPTION

The combination meter mirror ECU receives signals from the navigation receiver assembly via local bus communication and displays audio system and navigation system information on the headup display.

This DTC is stored when the combination meter mirror ECU cannot receive the signal.

DTC No.

Detection Item

DTC Detection Condition

Trouble Area

Memory

Note

B150A

Lost Communication with HMI-LAN

After the combination meter mirror ECU receives a registration information signal, which is sent by the navigation receiver assembly when the power switch is on (ACC), 1 or more times, the combination meter mirror ECU cannot receive the signal for 30 seconds or more.

  • Harness or connector
  • Navigation receiver assembly
  • Combination meter mirror ECU
