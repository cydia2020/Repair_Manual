Meter Illumination is Always Dark

DESCRIPTION

The combination meter assembly (meter circuit plate) receives signals from this circuit to adjust the illumination of the combination meter assembly (meter circuit plate). The combination meter assembly (meter circuit plate) sets the illumination level based on the user operation of the light control rheostat switch.

CAUTION / NOTICE / HINT

NOTICE:

Before replacing the main body ECU (multiplex network body ECU), refer to Registration.

HINT:

  • The meter illumination dims when the taillights are turned on.
  • Setting the meter illumination level to maximum brightness cancels the above dimming of the meter illumination.
  • Setting the meter illumination level to minimum brightness turns off the meter illumination.
  • The automatic light control sensor sensitivity can be customized.
