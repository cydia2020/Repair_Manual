Title: LIGHTING (EXT): LIGHTING SYSTEM (w/o AFS): Front Fog Light Circuit; 2016 - 2018 MY Prius [11/2015 - 12/2018]

Front Fog Light Circuit

DESCRIPTION

The main body ECU (multiplex network body ECU) controls the front fog lights.

WIRING DIAGRAM

CAUTION / NOTICE / HINT

NOTICE:

  • Inspect the fuses for circuits related to this system before performing the following procedure.
  • Before replacing the main body ECU (multiplex network body ECU), refer to Registration.
