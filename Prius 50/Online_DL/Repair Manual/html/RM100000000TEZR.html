Title: METER / GAUGE / DISPLAY: HEADUP DISPLAY: REASSEMBLY; 2016 - 2022 MY Prius [11/2015 -        ]

REASSEMBLY

PROCEDURE

1. INSTALL NO. 2 COMBINATION METER MIRROR

(a) Engage the 2 guides.

(b) Install the No. 2 combination meter mirror with the 3 screws.

2. INSTALL COMBINATION METER MIRROR

(a) Engage the 2 guides to install the combination meter mirror as shown in the illustration.

Install in this Direction

(b) Engage the 2 claws to install the mirror clip and combination meter mirror as shown in the illustration.

Install in this Direction

3. INSTALL COMBINATION METER WIRE

(a) Connect the connector.

(b) Engage the 2 clamps to install the combination meter wire.

4. INSTALL COMBINATION DISPLAY

(a) Engage the 2 guides as shown in the illustration.

Install in this Direction

(b) Install the combination display with the 2 screws.

5. INSTALL COMBINATION METER MIRROR ECU

(a) Engage the 2 guides.

Install in this Direction

(b) Connect the connector as shown in the illustration.

NOTICE:

Be sure to connect the connector securely.

(c) Push down the holder to engage the 2 claws as shown in the illustration.

NOTICE:

Be sure to engage the claws securely.

(d) Connect the 2 connectors to install the combination meter mirror ECU.

6. INSTALL NO. 1 COMBINATION METER MIRROR PLATE

(a) Install the No. 1 combination meter mirror plate with the 5 screws.

7. INSTALL NO. 1 COMBINATION METER MIRROR CASE

(a) Engage the 2 guides and 6 claws as shown in the illustration.

Install in this Direction

(b) Install the No. 1 combination meter mirror case with the screw.
