Title: LIGHTING (INT): VANITY LIGHT BULB: REPLACEMENT; 2016 - 2022 MY Prius [11/2015 -        ]

REPLACEMENT

CAUTION / NOTICE / HINT

HINT:

  • Use the same procedure for the RH side and LH side.
  • The following procedure is for the LH side.

PROCEDURE

1. SEPARATE VANITY LIGHT ASSEMBLY

(a) Using a moulding remover, disengage the claw to separate the vanity light assembly.

2. REMOVE VANITY LIGHT BULB

(a) Remove the vanity light bulb from the bulb holder as shown in the illustration.

Remove in this Direction

3. INSTALL VANITY LIGHT BULB

(a) Install the vanity light bulb to the bulb holder as shown in the illustration.

Install in this Direction

4. INSTALL VANITY LIGHT ASSEMBLY

(a) Engage the claw to install the vanity light assembly.
