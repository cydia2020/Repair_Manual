Title: BRAKE SYSTEM (OTHER): BRAKE BOOSTER PUMP: INSTALLATION; 2016 - 2022 MY Prius [11/2015 -        ]

INSTALLATION

PROCEDURE

1. INSTALL NO. 1 BRAKE TUBE CLAMP BRACKET

(a) Install the No. 1 brake tube clamp bracket to the brake booster pump assembly with the bolt.

Torque:

7.0 N·m {71 kgf·cm, 62 in·lbf}

2. INSTALL NO. 1 BRAKE ACTUATOR HOSE

(a) Install the No. 1 brake actuator hose to the brake booster pump assembly, and slide the clip to secure it.

*a

Alignment Mark

*b

Rib of Brake Booster Pump Assembly Port

Front of the vehicle

NOTICE:

  • When installing the No. 1 brake actuator hose, make sure to align its alignment mark with the rib of the brake booster pump assembly.
  • Install the clip within the range shown in the illustration.

3. INSTALL BRAKE BOOSTER PUMP ASSEMBLY

(a) Install the brake actuator case collar and brake booster pump bushing to the brake actuator bracket assembly.

(b) Temporarily install the brake actuator bracket assembly to the vehicle body.

NOTICE:

Do not apply excessive force to the brake lines or refrigerant lines.

(c) Temporarily install the brake booster pump assembly as shown in the illustration.

NOTICE:

  • Do not apply excessive force to the brake lines or refrigerant lines.
  • Do not drop the brake booster pump assembly when carrying it.
  • Be careful not to allow brake fluid to enter the connector.
  • If installing a new brake booster pump assembly, do not remove the hole plugs before connecting the brake lines because the brake booster pump assembly is filled with brake fluid.
  • Do not carry the brake booster pump assembly by the parts shown in the illustration.

HINT:

  • Move the brake actuator bracket assembly as necessary to allow the brake booster pump assembly to be temporarily installed.
  • Make sure to only temporarily install the brake booster pump assembly as the brake actuator assembly needs to be installed first.

(d) Install the brake actuator bracket assembly with the bolt.

Torque:

19 N·m {194 kgf·cm, 14 ft·lbf}

(e) Install the 2 nuts.

Torque:

19 N·m {194 kgf·cm, 14 ft·lbf}

NOTICE:

Tighten the nuts in the order shown in the illustration.

(f) Engage the clamp to install the No. 3 front brake tube to the brake actuator bracket assembly.

4. INSTALL ACCUMULATOR TO BRAKE MASTER CYLINDER TUBE

(a) Engage the 3 clamps to install the accumulator to brake master cylinder tube.

*a

Clamp Cover

-

-

(b) Engage the claw to install the clamp cover.

(c) Using a union nut wrench, connect the accumulator to brake master cylinder tube to the brake booster with master cylinder assembly.

Torque:

Specified tightening torque :

15.2 N·m {155 kgf·cm, 11 ft·lbf}

NOTICE:

  • Do not kink or damage the accumulator to brake master cylinder tube.
  • Do not allow the accumulator to brake master cylinder tube to twist or interfere with other parts or the vehicle body during tightening.
  • Do not allow any foreign matter such as dirt or dust to enter the accumulator to brake master cylinder tube from the connecting parts.

HINT:

  • Calculate the torque wrench reading when changing the fulcrum length of the torque wrench.
