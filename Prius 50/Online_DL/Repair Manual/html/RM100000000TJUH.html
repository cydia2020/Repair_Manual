Title: DOOR LOCK: POWER DOOR LOCK CONTROL SYSTEM: Only Back Door cannot be Opened; 2016 - 2022 MY Prius [11/2015 -        ]

Only Back Door cannot be Opened

DESCRIPTION

The main body ECU (multiplex network body ECU) receives signals from the back door opener switch assembly. When a signal received is the main body ECU (multiplex network body ECU) activates the back door lock motor.

WIRING DIAGRAM

CAUTION / NOTICE / HINT

NOTICE:

  • Before replacing the main body ECU (multiplex network body ECU), refer to Registration.
