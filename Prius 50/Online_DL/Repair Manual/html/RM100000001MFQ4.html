Title: NAVIGATION / MULTI INFO DISPLAY: NAVIGATION SYSTEM: B156C; Extension Module Disconnected; 2016 - 2022 MY Prius [11/2015 -        ]

DTC

B156C

Extension Module Disconnected

DESCRIPTION

The stereo component tuner assembly and the navigation receiver assembly are connected by the USB communication line.

When an USB communication error occurs between the stereo component tuner assembly and the navigation receiver assembly, this DTC will be stored.

DTC No.

Detection Item

DTC Detection Condition

Trouble Area

B156C

Extension Module Disconnected

Stereo component tuner assembly disconnected

  • No. 1 navigation wire
  • Stereo component tuner assembly
  • Navigation receiver assembly

WIRING DIAGRAM

CAUTION / NOTICE / HINT

NOTICE:

After replacing the stereo component tuner assembly of vehicles subscribed to pay-type satellite radio broadcasts, XM radio ID registration is necessary.
