Title: PARK ASSIST / MONITORING: ULTRASONIC SENSOR (for Front): INSTALLATION; 2016 - 2018 MY Prius [11/2015 - 12/2018]

INSTALLATION

PROCEDURE

1. INSTALL FRONT CORNER ULTRASONIC SENSOR RETAINER

HINT:

  • Perform this procedure only when replacement of the front corner ultrasonic sensor retainer is necessary.
  • If a front corner ultrasonic sensor retainer has been removed, replace it with a new one.
  • The illustration is for the LH side. The orientation for the RH side is the opposite of the LH side.

(a) Clean the surface of the front bumper cover.

(1) Remove any remaining double-sided tape from the front bumper cover.

(2) Wipe off any adhesive residue with cleaner.

(b) Cover the sensor installation hole with a 25 mm (0.984 in.) circular piece of tape.

*a

25 mm (0.984 in.)

(c) Using a brush or felt, apply primer or equivalent to the front corner ultrasonic sensor retainer installation area.

*a

Line

Primer

NOTICE:

  • Use a clean brush or felt.
  • Do not touch the front bumper assembly until the primer has dried.

(d) Peel off the release paper from the front corner ultrasonic sensor retainer.

(e) Align the front corner ultrasonic sensor retainer with the mark on the front bumper cover and install it as shown in the illustration.

NOTICE:

The double-sided tape of a front corner ultrasonic sensor retainer will deteriorate if it is detached. Make sure to use a new front corner ultrasonic sensor retainer when reattachment is necessary.

*a

Line

Press area

HINT:

Press the area shown in the illustration with a force of 30 N (3.0 kgf) for 3 seconds to securely install the front corner ultrasonic sensor retainer to the front bumper cover. Confirm there is no clearance between the front corner ultrasonic sensor retainer and front bumper cover.
