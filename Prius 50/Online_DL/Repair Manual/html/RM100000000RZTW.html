REASSEMBLY

PROCEDURE

1. INSTALL HOOD STAY BRACKET

(a) Engage the guide.

(b) Install the hood stay bracket with the bolt.

2. INSTALL HOOD SUPPORT ASSEMBLY

(a) Engage the 3 claws to install the hood support assembly.

3. INSTALL HOOD STAY HOLDER

(a) Engage the 2 claws to install the hood stay holder.

4. CONNECT WASHER HOSE ASSEMBLY

(a) Engage the grommet.

(b) Engage the 5 clamps to connect the washer hose assembly.
