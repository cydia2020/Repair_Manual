Title: LIGHTING (EXT): REAR COMBINATION LIGHT ASSEMBLY: REASSEMBLY; 2019 - 2022 MY Prius [12/2018 -        ]

REASSEMBLY

CAUTION / NOTICE / HINT

HINT:

  • Use the same procedure for the RH side and LH side.
  • The following procedure is for the LH side.

PROCEDURE

1. INSTALL NO. 2 REAR COMBINATION LIGHT LENS AND BODY

2. INSTALL STOP LIGHT LED GASKET

(a) Install a new stop light LED gasket to the stop light LED.

3. INSTALL TAIL LIGHT LED GASKET

(a) Install a new tail light LED gasket to the tail light LED.

4. INSTALL STOP LIGHT LED

(a) Turn the stop light LED as shown in the illustration to temporarily install it.

Install in this Direction

(b) Using a T20 "TORX" socket wrench, install the stop light LED with the screw.

5. INSTALL TAIL LIGHT LED

(a) Turn the tail light LED as shown in the illustration to temporarily install it.

Install in this Direction

(b) Using a T20 "TORX" socket wrench, install the tail light LED with the screw.

6. INSTALL REAR COMBINATION LIGHT LENS AND BODY

7. INSTALL REAR COMBINATION LIGHT SOCKET AND WIRE SUB-ASSEMBLY

(a) Engage the clamp to install the rear combination light socket and wire sub-assembly.

8. INSTALL REAR SIDE MARKER LIGHT BULB

(a) Install the rear side marker light bulb to the rear combination light socket and wire sub-assembly.

(b) Turn the rear combination light socket and wire sub-assembly with the rear side marker light bulb as shown in the illustration to connect them as a unit.

Install in this Direction

9. INSTALL REAR TURN SIGNAL LIGHT BULB

(a) Install the rear turn signal light bulb to the rear combination light socket and wire sub-assembly.

(b) Turn the rear combination light socket and wire sub-assembly with the rear turn signal light bulb as shown in the illustration to connect them as a unit.

Install in this Direction

10. INSTALL BACK-UP LIGHT BULB

(a) Install the back-up light bulb to the rear combination light socket and wire sub-assembly.

(b) Turn the rear combination light socket and wire sub-assembly with the back-up light bulb as shown in the illustration to connect them as a unit.

Install in this Direction
