15. TEMPORARILY INSTALL EGR VALVE ASSEMBLY WITH EGR COOLER

(a) Install a new EGR cooler gasket.

(b) Using an E8 "TORX" socket wrench, temporarily install the EGR valve assembly with EGR cooler with the 2 stud bolts.

Torque:

9.5 N·m {97 kgf·cm, 84 in·lbf}

(c) Temporarily install the bolt, 2 nuts (A) and 2 new nuts (B).

16. TEMPORARILY INSTALL EGR PIPE ASSEMBLY

(a) Install new inlet EGR gasket and EGR pipe gasket.

(b) Temporarily install the EGR pipe assembly with the 4 bolts.

19. INSTALL WATER OUTLET

(a) Install the water outlet and a new water outlet gasket to the cylinder head sub-assembly with the 2 nuts.

Torque:

28 N·m {286 kgf·cm, 21 ft·lbf}

(b) Connect the No. 9 water by-pass hose to the purge valve and slide the clip to secure it.

(c) Connect the No. 4 water by-pass hose to the EGR valve assembly with EGR cooler and slide the clip to secure it.
