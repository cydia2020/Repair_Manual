Additional Key cannot be Registered

DESCRIPTION

If additional registration is not possible, a malfunction of the electrical key transmitter sub-assembly, certification ECU (smart key ECU assembly), power switch, No. 2 indoor electrical key antenna assembly (front floor), hybrid vehicle control ECU or electrical key and tire pressure monitoring system receiver assembly is suspected.

CAUTION / NOTICE / HINT

NOTICE:

  • If registration of an electrical key transmitter sub-assembly cannot be performed, confirm the procedure in Immobiliser System.

    Click here

  • The smart key system (for Entry Function) uses the LIN communication system and CAN communication system. Inspect the communication function by following How to Proceed with Troubleshooting. Troubleshoot the smart key system (for Entry Function) after confirming that the communication systems are functioning properly.

    Click here

  • When using the Techstream with the power switch off, connect the Techstream to the DLC3 and turn a courtesy light switch on and off at intervals of 1.5 seconds or less until communication between the Techstream and the vehicle begins. Then select the vehicle type under manual mode and enter the following menus: Body Electrical / Smart Key. While using the Techstream, periodically turn a courtesy light switch on and off at intervals of 1.5 seconds or less to maintain communication between the Techstream and the vehicle.
  • Before replacing the certification ECU (smart key ECU assembly), hybrid vehicle control ECU or electrical key transmitter sub-assembly, refer to Precaution.
  • After repair, confirm that no DTCs are output.

HINT:

When additional registration is not possible, freeze frame data may be stored in the certification ECU (smart key ECU assembly).

If freeze frame data is available, it can be helpful in determining whether any parts were malfunctioning or not or narrowing down the suspected areas based on the table below.

Certification ECU Log of ID Code Registration

Parameter Name

Cause of Problem and Countermeasure

Failed Registration Mode

Make sure that another mode or a different registration mode is not selected, and then perform registration again.

Key Registration Max

Registration was not possible because the maximum number of the electrical key transmitter sub-assemblies has already been registered to the certification ECU (smart key ECU assembly).

If a new electrical key transmitter sub-assembly needs to be registered, such as when one of the registered electrical key transmitter sub-assemblies has been lost, erase an unnecessary ID code first.

Key Verification: Amp Abnormal

Although a registered electrical key transmitter sub-assembly was held near the power switch, electrical key transmitter sub-assembly registration could not be confirmed due to a malfunction of the immobiliser amplifier, such as a short to ground. DTC B278A or B2784 was stored.

Key Verification: Read Error

A registered electrical key transmitter sub-assembly was not held near the power switch properly or electrical key transmitter sub-assembly registration cannot be confirmed due to wave interference, etc.

Register the electrical key transmitter sub-assembly using the correct procedure or use another registered electrical key transmitter sub-assembly.

Transponder Key Amp Abnormal

Although an unregistered electrical key transmitter sub-assembly was held near the power switch, electrical key transmitter sub-assembly registration cannot be completed due to a malfunction of the immobiliser amplifier, such as a short to ground. DTC B278A or B2784 was stored.

New Key Registration Error

An unregistered electrical key transmitter sub-assembly was not held near the power switch properly or electrical key transmitter sub-assembly registration cannot be confirmed due to a malfunction, such as wave interference.

Reregister the electrical key transmitter sub-assembly using the correct procedure or use another registered electrical key transmitter sub-assembly.

New Key Vehicle ID Error

Although the vehicle ID stored in the certification ECU (smart key ECU assembly) should be registered to the electrical key transmitter sub-assembly during electrical key transmitter sub-assembly registration, it was not successful.

Use another unregistered electrical key transmitter sub-assembly for registration.

New Key Registration Failed

Although an unregistered electrical key transmitter sub-assembly was held near the power switch for registration, electrical key transmitter sub-assembly registration was not successful due to wave interference (such as radio waves emitted from a device in the cabin) or a low transmitter battery.

New Key Already Registered

As a registered electrical key transmitter sub-assembly was held near the power switch by mistake, electrical key transmitter sub-assembly registration was not successful.

Using an unregistered electrical key transmitter sub-assembly, perform the registration procedure again.

New Key ID Code Error

Replace the certification ECU (smart key ECU assembly).

New Key Information Abnormal

The ID code of an electrical key transmitter sub-assembly could not be registered to the certification ECU (smart key ECU assembly). If 2 electrical key transmitter sub-assemblies are held near the power switch at the same time, incorrect registration information will be stored in the certification ECU (smart key ECU assembly), suspending registration.

Reregister the electrical key transmitter sub-assembly using the correct registration procedure.
