INSTALLATION

PROCEDURE

1. INSTALL CHILD RESTRAINT SEAT ANCHOR BRACKET SUB-ASSEMBLY LH (for LH Side)

(a) Temporarily install the child restraint seat anchor bracket sub-assembly LH with the 2 bolts.

(b) Fully tighten the bolt (A), and then the bolt (B) to install the child restraint seat anchor bracket sub-assembly LH.

Torque:

19.5 N·m {199 kgf·cm, 14 ft·lbf}

(c) Engage the clamp.
